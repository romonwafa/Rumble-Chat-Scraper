# Rumble Live Chat Scraper

Usage: `python3 scraper.py [options] [URL]`

## Current features:
- Scrapes existing live chats and details associated with each chat message:
	- Username
	- User ID
	- Message ID
	- Avatar URL
	- Profile URL
	- Message Text
- Output csv file
